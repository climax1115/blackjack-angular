Basic Strategy guide

Disclaimer: Basic Strategy is not enough to beat the House edge even if you get lucky from time to time. The House has a huge advantage in that the players always play first, meaning if a player goes bust, it is an automatic lose for the player before the dealer even plays. Whilst basic strategy provides a good guideline, you will need counting, deviations, true count conversions and betting strategy to actually beat the game of black jack.

What is Basic Strategy?

Basic strategy is a set of suggested actions at each scenario derived from a computer simulation. A simulation was set up where a computer played several hundreds of million hands of blackjack and recorded what happened. Through trial and error, it determined which actions have the greatest odds of success given every possible combination of starting hand and dealer up card. Despite playing the "Best action" for each hand, the player will still lose money to the House over time due to the House's edge. If you are curious about the calculations, check out Mike Shakleford's video where he builds Basic strategy from scratch using excel.
